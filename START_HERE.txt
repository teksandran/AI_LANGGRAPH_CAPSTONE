--------------------------------------------------------------------------------
STEP 1: Start the Backend (Flask API)
--------------------------------------------------------------------------------

1. Open Command Prompt/Terminal

2. Navigate to project:
   cd "C:\New Development 2025\AI ML\Agentic AI\Capstone Demo\yelp_mcp"

3. Start the server:
   python api_server.py

4. You should see:
   ============================================================
   Multi-Agent API Server
   ============================================================
   LLM Provider: OPENAI
   SupervisorAgent: [OK] Initialized
   RAG System: [OK] Ready
   ============================================================

   Server starting on http://localhost:5000

âœ… KEEP THIS TERMINAL OPEN - The server must stay running!

--------------------------------------------------------------------------------
STEP 2: Index Products (One-time setup)
--------------------------------------------------------------------------------

1. Open a NEW Command Prompt/Terminal

2. Run this command:
   curl -X POST http://localhost:5000/api/index-products

3. Wait 2-3 minutes (it's scraping Botox and Evolus websites)

4. You'll see:
   {
     "status": "success",
     "message": "Products indexed successfully"
   }

âœ… You only need to do this ONCE (unless you delete the index)

--------------------------------------------------------------------------------
STEP 3: Test the API
--------------------------------------------------------------------------------

Test if everything works:

curl -X POST http://localhost:5000/api/query -H "Content-Type: application/json" -d "{\"query\": \"What is Botox?\"}"

You should get a detailed response about Botox!

--------------------------------------------------------------------------------
STEP 4: Set Up React Frontend
--------------------------------------------------------------------------------

1. Navigate to your React project:
   cd "C:\path\to\your\react-project"

2. Install axios:
   npm install axios

3. Create API service file:
   - Create folder: src/services/
   - Create file: src/services/apiService.js
   - Copy code from REACT_API_GUIDE.md

4. Update App.jsx:
   - Copy the updated App.jsx code from REACT_API_GUIDE.md

--------------------------------------------------------------------------------
STEP 5: Start React App
--------------------------------------------------------------------------------

1. In your React project folder:
   npm start

2. React app opens at:
   http://localhost:3000

3. Try asking:
   - "What is Botox used for?"
   - "Compare Botox and Evolus"
   - "Find Botox clinics in New York"
